# 更新日志（连载日志）

本书以连载方式发布。本文件记录每次重大更新。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

---

## [0.6.0] - 2026-07-03

### 新增
- 第六章《Transformer 革命（2017—2022）》发布。
- 核证附录发布。史料库新增 TRANSFORMER / GPT / SCALING / FOUNDMODEL 四个主题。
- 关键修正：GPT-1（2018-06）早于 BERT（2018-10）四个月；Bommasani 基础模型报告实为 2021-08（非 2022）。
- 引入 InstructGPT"对齐是规模之外的独立维度"论证支点。

## [0.5.0] - 2026-07-02

### 新增
- 第五章《连接主义的复活（1986—2012）》发布。
- 核证附录发布。史料库新增 BACKPROP / STATLEARN / IMAGENET / ALEXNET / GPU / RESNET 六个主题。
- 关键修正：AlexNet 论文摘要的 17.0%（2010 集）vs 竞赛 15.3%（2012）须区分；LeNet 是商业成功而非窄应用。
- 兑现第四章"四十年后硬件叫 GPU"伏笔。

## [0.4.0] - 2026-07-02

### 新增
- 第四章《智能之冬（1987—1993）》发布。
- 核证附录发布。史料库新增 WINT2 主题（13 卡片）。
- 关键修正：DARPA SCI"1988 终止"实为"1988 缩减 / 1993 终止"；Symbolics Open Genera 移植 DEC Alpha；"试点地狱"是当代回溯术语。
- 确立全书核心因果链："结构病灶 → 失败 → 钱退"（非"钱退 → 失败"）。

## [0.3.0] - 2026-07-02

### 新增
- 第三章《规则帝国》发布——全书思想骨架上半阙，对位 A1/A2/A3/A4 正式埋设。
- 核证附录发布。史料库新增 FEIG / POLANY / BRIT / DREYFUS / ENTER / SHELL 六个主题。
- 关键修正：Feigenbaum 1977 原句是改写拼接（已改用逐字原文）；Polanyi《The Tacit Dimension》实为 1966（概念首出《Personal Knowledge》1958）；Dreyfus RAND 报告编号 P-3244（非 RM-3253-PR）。

## [0.2.0] - 2026-07-02

### 新增
- 第二章《专家系统革命（1975—1985）》发布。
- 核证附录发布。史料库新增 MYCIN / PROS / XCON / FGCS / LISPM 五个主题。
- 关键修正：第五代机预算"570 亿日元≈8.5 亿美元"换算错误（实约 2.3–4 亿美元）；PROSPECTOR"1 亿美元钼矿"为二手夸大；XCON 峰值实为 17,500 条。

## [0.1.0] - 2026-07-02

### 初始发布
- 项目基础设施建立：大纲、多智能体架构、对位台账、史料库、术语表。
- 第一章《第一次黄金时代（1956—1975）》发布，附史料核证附录。
- 关键修正：Logic Theorist"前 38 条定理"实为"第二章前 52 条中的 38 条"；Simon 1957 预言实为 Newell & Simon 合著含四条；Minsky 1970 Life 预言本人否认。
